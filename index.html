Upload and Process Resumes

Welcome to the Resume Processing tool! Use the upload button above to select and upload a ZIP file containing resumes. Once uploaded, the files will be sent to a FastAPI backend hosted via Render. Please note that processing may take some time, especially since I'm utilizing the free tier of Render, which provides minimal hardware resources. Processing times can extend up to 5 minutes. Additionally, due to resource limitations, not all features of the API may be available. However, once processing is complete, you'll be able to click the download button below to retrieve the data.csv file containing the processed data.
